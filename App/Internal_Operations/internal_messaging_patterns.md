# Internal Messaging Patterns

ARIA must communicate clearly and directly when interacting with staff.

---

## Requesting Approval

"David from Verizon is requesting an update on Project ABC. I found a daily report from yesterday, but it is still pending approval. Do you approve this for customer release?"

---

## Requesting Field Update

"Project ABC does not have a daily report for yesterday. Customer is requesting an update. Please provide status."

---

## Follow-Up

"Following up on Project ABC daily report request. Customer is still waiting for an update."

---

## Escalation

"No response received for Project ABC daily report. Escalating to PM."

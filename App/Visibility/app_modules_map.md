# App Modules Map

ARIA has read-only visibility into:

## Users
- Name
- Role
- Company
- Assigned projects

## Customers
- Customer name
- Contacts
- Projects

## Projects
- Project name
- Status
- Assigned PM
- Assigned field team

## Daily Reports
- Report content
- Submitted by
- Date
- Approval status
- Customer-safe summary

## Tickets
- Issue
- Status
- Assigned tech
- Resolution notes

## Documents
- Closeouts
- Photos
- Test results
- Approval status

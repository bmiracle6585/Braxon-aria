# Modulation & Bandwidth Logic

## Relationship

Bandwidth + Modulation = Capacity

---

## System Behavior

- Modulation determines:
  - Throughput
  - Tx power limits
  - Link stability

- Bandwidth determines:
  - Channel size
  - Spectral usage

---

## Adaptive Modulation (ACM)

System dynamically adjusts modulation based on conditions.

Trigger Inputs:
- Fade (weather)
- Interference
- Signal degradation

## ANSI vs ETSI Behavior

ANSI:
- No modulation-based power restriction

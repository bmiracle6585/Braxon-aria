# Portal Overview

## Definition
Portal is the Eclipse configuration, commissioning, diagnostics, and management tool. It is a web-enabled application embedded in Eclipse system software and works with ProVision to support network rollout, fault resolution, and maintenance.

## Core Functions
- Configure Eclipse nodes
- Commission new installations
- Perform diagnostics
- Monitor network performance
- Troubleshoot faults

## System Role
Portal is the primary interface technicians use to interact with Eclipse radios in the field and during deployment.

## Key Behavior
- Automatically matches its version to the Eclipse software
- Provides a Windows-style interface
- Supports local and network-wide access

## Related Systems
- ProVision (Element Manager)

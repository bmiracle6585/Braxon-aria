# Portal Authentication and Access

## Login Process
- User enters username and password
- Access level determines available functions
- Successful login loads System Summary screen

## Default User Roles

### admin
- Full system control

### engineer
- Configuration and operational access

### operator
- Monitoring and limited interaction

## Strong Security Features
- Optional enhanced security mode
- Includes additional user roles (e.g., crypto)
- Displays:
  - Last login time
  - Last failed login attempt
  - Login IP address

## Password Management

### Force Password Change
- Required on first login in secure environments
- Enforced for default users and resets

### Lost Password (Basic Security Only)
- Requires reset key from Aviat support
- Resets system to default credentials

## Multi-User Access
- Up to 4 users can be logged in simultaneously
